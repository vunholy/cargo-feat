# Security Policy

## Supported Versions

The following versions of **cargo-feat** are currently receiving security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| 0.1.x   | :x:                |
| < 0.1   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in `cargo-feat`, please report it privately.

**Where to report:**
- Open a private security advisory on GitHub:
  https://github.com/vunholy/cargo-feat/security/advisories
- Or contact: itzd7m51@gmail.com

**What to expect:**
- You will receive an initial response within a reasonable time frame (usually within a few days)
- If confirmed, a fix will be prioritized and released as soon as possible
- You will be credited if the report leads to a valid security fix (unless you request otherwise)

Please do not disclose vulnerabilities publicly until a fix has been released.
